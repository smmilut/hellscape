goal :

attack enemies to send them up into spikes

input :

keyboard or gamepad with following bindings :

  • move : arrow keys, or A/D, or Q/D, or gamepad directions
  • jump (and double jump) : space bar, or J, or gamepad button A
  • attack : h, or gamepad button X
  • start : Enter, or gamepad button Start
